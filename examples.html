Examples

To give you an idea of how client-side applications are built using Chaplin, here are some projects and example applications whose source is freely accessible.

Ost.io

Ost.io is an open-source forum for GitHub projects (code). It uses Brunch with Chaplin as an application skeleton and Rails for the backend.

Todos

Todos (code) is a port of the famous Backbone todo example app. It uses localStorage to persist data.

Alternative versions:

  • Symfony TodoMVC Bundle (code) — based on Symfony PHP framework.

Composing with Chaplin

Composing with Chaplin is an example that shows how Chaplin regions and compositions behave. Find its source code on GitHub.

Facebook Like Browser

This example application uses Facebook client-side authentication to display the user’s Facebook Likes. Find its source at github.com/chaplinjs/facebook-example.

Tweet your Brunch

github.com/brunch/twitter is a simple twitter client. It uses Twitter client-side authentication to display the user’s feed and create new tweets.

In Production

Below you find some of the companies and projects using Chaplin in production. Are you using Chaplin too? Go ahead and add your application to the wiki page to let us know!

Delicious

Delicious is a free service designed to be the best place to save what you love on the web, whether it's a video, picture, product, blog post, article or music. We then help you remember and find it later. It’s using Brunch as application builder and Chaplin as a framework on top of Backbone.

GED VIZ by Bertelsmann Stiftung

GED VIZ lets you create and share visualizations of global economic relations - for research, teaching and storytelling. Source code is available on GitHub

Bank of Ireland

The Bank of Ireland Fund Centre is a free service by MoneyMate that allows users to check funds, daily prices and performance.

Moviepilot

Moviepilot is a home for upcoming movies, where you can discover and read about the movies that interest and excite you.
